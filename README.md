# culinary-hub
Culinary Hub is a scalable Django-based restaurant web platform that integrates secure user authentication, reservation services, profile management, contact handling, and a dynamic blog system, delivering a seamless and engaging digital dining experience.

✨ Key Features
🔐 Secure user authentication system (registration, login, logout) ,
👤 User profile management for personalized experience,
📅 Table reservation system for booking services,
📩 Contact form for user queries and communication,
📝 Dynamic blog section (list & detailed blog posts using slug URLs),
🌐 Clean, responsive, and interactive user interface

🛠 Tech Stack
Backend: Django (Python),
Frontend: HTML, CSS, JavaScript,
Database: SQLite
